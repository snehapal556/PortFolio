# PortFolio
This is a personal portfolio website built using HTML5, CSS3, and JavaScript to showcase my skills, projects, and contact information. The website features a clean, modern, and responsive design, ensuring a seamless experience across all devices.

Key Features:

HTML5: Provides the structure and semantic layout for the website.

CSS3: Implements styling, layout, and responsive design elements.

JavaScript: Adds dynamic functionality such as form validation, interactive components, and smooth user experience.

The website is designed to give a clear overview of my technical skills and projects, making it easy for visitors and potential employers to explore my work.
